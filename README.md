# CyberShop Catálogo

Reemplaza en GitHub estos archivos:

- index.html
- app.js
- productos.js
- styles.css

No borres la carpeta logos ni tus imágenes.

## Cambiar WhatsApp

En productos.js:

```js
whatsappNumber: "57TU_NUMERO_AQUI",
```
